Extract icons and cursors from program files and DLL's

It is recommended to read this help article about icon libraries first to understand what icon and cursor resources are.

  1. Choose File|Open and select the EXE, DLL, ICL or other executable file you want to extract the resources from.

    A library window appears with all found icons and cursors listed. Of course, the list can be blank if the file does not contain any resources. Note that 16-bit executables are not yet recognized.

  2. Select the resources you want to extract.

    You may hold down Shift or Ctrl to select multiple items.

  3. Choose Library|Extract and save or click the appropriate speed button, and select a folder for the extracted resources.

    You will get individual ICO and CUR files in the selected folder which contain extracted resource data.
