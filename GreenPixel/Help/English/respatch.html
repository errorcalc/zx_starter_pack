Hack executables in order to replace hard-wired icons and cursors

With the resource editing capability of GreenPixel, it is easy to replace resources in EXE files in order to modify certain icons and cursors used by applications. However, this can cause programs not to work - they may recognize changes and attribute them to a virus attack - and is often prohibited by the license agreement claiming that you should not modify the file contents. Be careful what you do and always create a backup of the unmodified program file!

  1. Open the EXE or DLL file you want to hack.
  2. Select the resource to replace and choose Library|Replace resources or click the appropriate speed button.
  3. Select the icon or cursor file you want to replace the resource with. Be careful you replace an icon with an ICO and a cursor with a CUR, otherwise the program will probably not work.
  4. Choose File|Save to save the changes you made.
