Uninstalling GreenPixel properly:

If you used an installer program to set up GreenPixel, you have to select Control Panel|Programs and Features, browse for GreenPixel in the list appearing, and choose Uninstall.

If you unpacked a portable version of GreenPixel, you have to:

  1. Make sure that no file types are associated with GreenPixel. Uncheck all previously set associations at Settings|Preferences|Install.
  2. Delete the program folder.
